# Tarea-semana-11
Cultura digital
Ejemplo 1
Situación: Respirar.
Por qué no es necesaria una condicional: Siempre respiramos para vivir. No depende de “si pasa algo”.
Ejemplo 2
Situación: El sol sale cada mañana.
Por qué no es necesaria una condicional: El sol saldrá sin importar lo que hagamos o lo que pase.

Ejemplo 3
Situación: Contar del 1 al 10.
Por qué no es necesaria una condicional: La secuencia numérica siempre será igual, no depende de una condición.
Ejemplo 4
Situación: Beber agua para quitar la sed.
Por qué no es necesaria una condicional: Beber agua siempre ayuda a calmar la sed, no depende de otra acción o decisión.
